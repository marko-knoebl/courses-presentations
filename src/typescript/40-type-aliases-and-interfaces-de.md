# Type-Aliases und Interfaces

## Type-Aliases und Interfaces

_Type-Aliases_: ermöglichen das speichern einer Typendeklaration unter einem Namen

_Interfaces_: ähnliche Funktionalität, andere Syntax, etwas eingeschränktere Funktionalität
